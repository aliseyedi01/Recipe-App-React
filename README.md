## Run Locally

1. Clone the project

```bash
  https://github.com/aliseyedi01/Recipe-App-React.git
```

2. Go to the project directory

```bash
  cd Music-App-React
```

3. Install dependencies

```bash
  npm install
```

4. Start the server

```bash
  npm run start
```
